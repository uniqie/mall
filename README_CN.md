## 简介
这一切的开始，都要源于课程老师布置的任务。本着希(wang)望(xiang)锻炼我们的综合能力。老师希望我们能开发一个前端项目，完整地实现商城功能（包括主页、商品添加购物车、商品列表展示、个人资料页面展示）。于是本仓库便被建立起来了。

简要介绍一下，本项目主要使用到的技术框架为[taro](https://docs.taro.zone/docs/README/ "taro")和react。
